# Accessible Tabs Code Challenge

### Pre-requisites:
- Node v14 or later

## Running locally
1. `npm install`
2. `npm run start`
3. Open http://localhost:8080

 
---
 
## Running tests
1. if you haven't done so, `npm install`
2. `npm test` 

    2a. `npm test --watch` to run in interactive watch mode
